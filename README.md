# 📊 Aryan Prasad's Portfolio

My personal portfolio website to showcase my projects, skills, and journey as a Bachelor of Computer Applications (BCA) student and Aspiring Data Analyst. It is built using pure HTML, CSS, and vanilla JavaScript (no external libraries or frameworks).

* **Live Link:** [thearyanprasad.github.io](https://thearyanprasad.github.io)
* **Resume:** [aryan_resume.pdf](./aryan_resume.pdf)

---

## ✨ Features

* **Custom Cursor:** A smooth dual-circle mouse pointer animation that follows your movement.
* **Dark/Light Mode:** A simple theme toggle that saves your choice in the browser using `localStorage`.
* **SVG Texture:** A subtle hand-drawn paper/noise effect using custom SVG turbulence filters.
* **Scroll Animations:** Smooth fade-up reveals as you scroll through the sections.
* **Contact Form:** A modal form connected to the Web3Forms API to send messages without refreshing the page.

---

## 🛠️ Built With

* HTML5 for semantic page layout.
* CSS3 for styling, responsive grid structure, and keyframe animations.
* JavaScript (ES6) for the custom cursor tracking, ScrollSpy active navigation, and theme toggling.

## 💻 How to Run Locally

Since this is a vanilla site, you can run it without any package installations:

1. Clone the repo:
   ```bash
   git clone https://github.com/thearyanprasad/Portfolio.git
   ```
2. Go to the project directory:
   ```bash
   cd Portfolio
   ```
3. Open `index.html` in your browser, or run a local server (like the VS Code Live Server extension).
